# 🎹 MIDI Web

A web application for working with MIDI input: recording, visualization, and future analysis (including AI-assisted features).

---

## 🚀 Tech Stack

- React + TypeScript
- Vite
- Jest + Testing Library
- Node.js + npm

---

## ⚙️ Requirements

- Node.js: 24.14.0
- npm: 11.9.0

Check versions:

```bash
node -v
npm -v
```

---

## 📦 Installation

```bash
npm install
```

or (recommended for clean installs / CI):

```bash
npm ci
```


---

## 🧑‍💻 Development

Start dev server:

```bash
npm run dev
```

---

## 🏗️ Build

```bash
npm run build
```

---

## 🧪 Testing

Run tests:

```bash
npm test
```

Run tests with coverage:

```bash
npm run coverage
```
